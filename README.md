# 2025YaleQHack
This is the soluation of QuBruin team for 2025 Yale Hackathon. 




# Environment setup

Windows

```bash
cd path\to\your\project
py -m venv yalehack
yalehack\Scripts\activate
pip install -r requirements.txt
```

Mac

```bash
cd path/to/your/project
python3 -m venv yalehack
source yalehack/bin/activate
pip install -r requirements.txt
```
